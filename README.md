Button of Activation
====================

What is the Button of Activation?
---------------------------------

The **Button of Activation** is a command-line text adventure built in Java. You are Henchman Qess. Omega, tasked with repeatedly challenging a compromised security console using cryptic commands to complete the **Breach Protocol**​, a protocol that turns the Sun into a Red Giant.

Each stage presents a randomly generated challenge—from cracking security ports and cracking math problems to manipulating file systems—all while racing against the clock. Can you break the console before it manages to stabilize?

### Key Features

*   **Procedural Challenges:** Each playthrough features a unique sequence of challenges..
    
*   **Command Line Interface:** A classic, immersive terminal experience.
    

How to Play (For Users/Players)
-------------------------------

There are two ways to run the game on your system:

### 1. Windows (Recommended)

*   **Download:** Download the **Button of Activation.exe** file from the Downloads section.
    
*   **Run:** Simply double-click the file to launch the game instantly. _(It includes the Java Runtime Environment and does not require a separate Java installation.)_
    

### 2. Mac, Linux, and Universal

To run the game directly from the source code, you must have the **Java Development Kit (JDK) version 8 or higher** installed.
```bash
git clone https://github.com/Ravis-World/Button-of-Activation.git
cd Button-of-Activation
cd src
javac \*.java
java -cp . ButtonOfActivationGame
```
